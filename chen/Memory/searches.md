# חן — לוג חיפושים

זהו הזיכרון הפרסיסטנטי של חן. כל חיפוש שהיא מבצעת מתועד כאן בפורמט קבוע.
חן בודקת את הקובץ הזה (באמצעות `Grep`) לפני כל חיפוש חדש כדי למנוע כפילויות בנושאים יציבים.

## פורמט קבוע לכל entry

```
## YYYY-MM-DD HH:MM | <נושא החיפוש> [DYNAMIC?]
**מילות מפתח:** keyword1, keyword2, keyword3
**שאילתות שנעשו:** "query 1 exact text", "query 2 exact text"
**מקורות שנמצאו:**
- [כותרת מקור](URL) - איכות: ⭐⭐⭐⭐ - <הערה קצרה>
- [כותרת מקור](URL) - איכות: ⭐⭐⭐ - <הערה קצרה>
**נבחר:** <המקור הנבחר ולמה>
**קובץ ב-Content:** <filename>.md
---
```

ה-`[DYNAMIC]` בכותרת אם הנושא דינמי (חדשות, מחירים, סטטיסטיקות, "השבוע/החודש"). יציב = בלי תג.

## כללי שימוש

1. **append בלבד** — entries חדשים בסוף הקובץ. אל תמחקי ישנים.
2. **גרפ קודם** — לפני כל חיפוש, `Grep` את מילות המפתח של הבקשה.
3. **30 ימים** — חיפוש שנעשה לפני יותר מ-30 ימים נחשב מיושן גם אם הנושא יציב.
4. **שני נושאים שונים** = שני entries נפרדים, גם אם נעשו באותו רגע.
